RFID Asset Tracking System

This project demonstrates a full-stack RFID tracking solution built with Spring Boot and React.

Backend API Development

We designed and implemented RESTful endpoints to manage RFID tags and scan logs:

  • POST /api/tags — Create new RFID tags
  • GET /api/tags — Retrieve all tags
  • GET /api/tags/{id} — Fetch tag by ID
  • POST /api/scan — Log manual scans
  • POST /api/scan/auto — Simulate scans via SerialPortReader

Explore the API using Swagger UI.

Tag Label Testing

We performed simulated tag label testing using mock serial port data. This allowed us to validate:

  • Tag ID recognition and location mapping
  • Scan logging and timestamp accuracy
  • Lifecycle updates for active/inactive assets

Lifecycle Management

We managed the entire RFID software development lifecycle:

  • Project Planning: Defined scope, endpoints, and data models
  • Architecture Design: Modular Spring Boot layers (model, repository, service, controller)
  • Software Development: Built secure, testable REST APIs with Swagger documentation
  • Testing: Verified endpoints via Swagger and Postman, seeded sample data for validation

Frontend (React + Material UI)

Responsive dashboard with:

  • Tag summary cards (total, active, inactive)
  • Bar chart of tag activity by location
  • Table of recent assets
  • Navigation to Swagger, GitHub, Postman

Built with:

  • React Router
  • Axios for API calls
  • Material UI components
  • Chart.js or Recharts for data visualization
