#  RFID Asset Tracking System
- This is a real-time tracking systems that Logs tag reads with timestamp and location, expose via REST
- It is suitable for monitoring and managing production process, purchased product and asset list.

## Tech Stack
- Spring Boot, React, MySQL
- Railway (backend), Vercel (frontend)

##  Modules
- Tag Registration
- Reader Integration
- Real-Time Tracking

##  Architecture
- RFID reader → Spring Boot API → MySQL
- React frontend → REST API → Live dashboard

##  Deployment
- Backend: Railway
- Frontend: Vercel
- DB: Railway MySQL
